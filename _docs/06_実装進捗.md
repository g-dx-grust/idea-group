# 実装進捗

更新日: 2026-04-25

## 今回実装したもの

資料だけの状態から、Next.js で起動できる MVP を作成しました。

| 領域 | 状態 |
|---|---|
| Next.js アプリ基盤 | 実装済み |
| デザイントークン | `src/app/globals.css` に定義 |
| データ保存 | `data/idea-store.json` のファイル保存 |
| ダッシュボード | 進行中、未請求、未入金、期限、最近更新を表示 |
| 現場一覧 | 検索、業務区分、ステータス、担当フィルタを実装 |
| 現場新規作成 | Server Action で作成し、JSON に保存 |
| 現場詳細 | 基本、確定測量、分筆測量、土地、立会申請、報酬計算、帳票タブを実装 |
| 土地明細 | 追加と一覧表示を実装 |
| 立会申請 | 追加と一覧表示を実装 |
| 報酬額計算書 | 明細入力、税計算、保存、集計表示を実装 |
| 帳票ビュー | 現場一覧、受託簿、見積書、請求書、報酬額計算書のHTML印刷ビューを実装 |
| UI 整え | Noto Sans JP、控えめな配色、サイドバー開閉、アクティブ表示、モバイル下部ナビ、簡易チュートリアルを実装 |
| テスト | 報酬計算、和暦変換の単体テストを追加 |

## 検証結果

```bash
pnpm typecheck
pnpm test
pnpm build
```

いずれも成功。

開発サーバー:

```bash
pnpm dev
```

`3000` が使用中だったため、`http://localhost:3001` で起動。

補足: ブラウザ自動確認用の `agent-browser` CLI はこの環境に入っていなかったため、HTTP 応答と開発サーバーログで代替確認。
