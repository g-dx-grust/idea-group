# 帳票テンプレート仕様書

バージョン: v0.1 / 2026-04-24

## 帳票生成方式

| 帳票 | 生成方式 | 出力 |
|---|---|---|
| 御見積書 | HTML → Puppeteer PDF | PDF |
| 御請求書 | HTML → Puppeteer PDF | PDF |
| 報酬額計算書 | HTML → Puppeteer PDF | PDF |
| 領収書 | HTML → Puppeteer PDF | PDF |
| 進捗状況報告書 | HTML → Puppeteer PDF + メール添付 | PDF |
| 立会申請各帳票 | docxtemplater（既存 .docx に差込） → PDF変換任意 | DOCX / PDF |
| 受託簿 / 工程表 | HTML → Puppeteer PDF | PDF |

```
伝票日付,借方科目,借方補助,借方金額,貸方科目,貸方補助,貸方金額,摘要
```

## 8. 実装メモ

### Puppeteer PDF 生成の共通設定
```ts
const pdf = await page.pdf({
    format: 'A4',
    printBackground: true,
    margin: { top: '15mm', bottom: '15mm', left: '15mm', right: '15mm' },
});
```

### docxtemplater の利用パターン
```ts
import Docxtemplater from 'docxtemplater';
import PizZip from 'pizzip';
import fs from 'fs';

const content = fs.readFileSync('templates/委任状.docx', 'binary');
const zip = new PizZip(content);
const doc = new Docxtemplater(zip, { paragraphLoop: true, linebreaks: true });
doc.render({ surveyorName: '加藤 和彦', ... });
const buf = doc.getZip().generate({ type: 'nodebuffer' });
```

### 和暦変換ユーティリティ
```ts
// 令和: 2019年〜
export function toWareki(date: Date): string {
    const y = date.getFullYear();
    const m = date.getMonth() + 1;
    const d = date.getDate();
    if (y >= 2019 && (y > 2019 || m >= 5)) {
        return `令和${y - 2018}年${m}月${d}日`;
    }
    if (y >= 1989) return `平成${y - 1988}年${m}月${d}日`;
    return `${y}年${m}月${d}日`;
}
```

### 印影画像の重ね方（Puppeteer）
```html
<!-- 見積書 HTML テンプレート内 -->
<div class="stamp-area">
    <img src="{{stampImageBase64}}" class="stamp-image" />
</div>
```
S3 から画像を `base64` に変換してテンプレートに埋め込む（Puppeteer がローカルファイルにアクセスできない環境向け）。
